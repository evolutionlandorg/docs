# 通证

进化星球是一个奇妙的通证星球。所有资产都以玩家持有的通证表示。它们分为两类：

* **FT** \(fungible tokens aka. ERC20 / TRC20\) 
* **NFT** \(non - fungible tokens aka. ERC721 / TRC721\).
